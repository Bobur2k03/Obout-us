Obout-us
О проекте / About the Project

Проект «Obout-us» — персональный сайт-портфолио, написанный на HTML5, CSS3 и JavaScript. Он включает адаптивный дизайн, анимации и иконки Font Awesome, а также разделы «Обо мне», «Проекты», «Контакты» и «Часто задаваемые вопросы».

"Obout-us" is a personal portfolio website built with HTML5, CSS3, and JavaScript. It features a responsive design, animations, and Font Awesome icons, and includes "About Me", "Projects", "Contacts", and "FAQ" sections.

Технологии / Technologies

HTML5 / CSS3 / JavaScript — основные веб-технологии для построения сайта, структуры и логики / core web technologies for building the site, its structure and logic.

Адаптивная вёрстка (Responsive Design) — обеспечивает корректное отображение сайта на разных устройствах / ensures correct display on various devices.

Анимации (Animations) — плавные эффекты для улучшения пользовательского опыта / smooth effects to enhance user experience.

Font Awesome — библиотека векторных иконок для интерфейса / a library of vector icons used in the interface.

Установка / Installation

Склонируйте репозиторий: git clone [https://github.com/username/obout-us.git.](https://github.com/Bobur2k03/Obout-us.git) Clone the repository.

Перейдите в папку проекта: cd obout-us. Navigate to the project folder.

Откройте файл index.html в браузере. Open the index.html file in your browser.
